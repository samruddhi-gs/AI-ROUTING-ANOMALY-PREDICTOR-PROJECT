# AI-Powered Network Anomaly Detector
An Intelligent system to detect BGP Hijacking and Network Anomalies using Machine Learning.

## Features
- **AI Backend:** Python Flask server using Random Forest Classifier.
- **Dataset:** Trained on NSL-KDD dataset.
- **Real-time Dashboard:** Built with HTML, CSS, and Chart.js.

## How to Run
1. Install requirements: `pip install flask flask-cors pandas scikit-learn joblib`
2. Start the Backend: `python app.py`
3. Open `dashboard.html` in your browser."# AI-ROUTING-ANOMALY-PREDICTOR-PROJECT"
